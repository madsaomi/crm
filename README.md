# 🎓 CRM Учебного Центра

Система управления учебным центром — полнофункциональное веб-приложение для автоматизации работы образовательного учреждения.

## 🛠 Технологии

| Компонент | Технология |
|-----------|-----------|
| Backend | Python 3.10+, Flask |
| ORM | SQLAlchemy |
| База данных | SQLite |
| Frontend | Bootstrap 5, Bootstrap Icons |
| Темы | Светлая / Тёмная (переключатель) |

## 🚀 Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd crm

# 2. Создать виртуальное окружение
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. (Опционально) Заполнить тестовыми данными
python seed.py

# 5. Запустить приложение
python app.py
```

Приложение будет доступно по адресу: **http://localhost:5000**

## 📦 Модули

### 👥 Студенты (`/students`)
- Список всех студентов с поиском
- Создание, редактирование, удаление
- Профиль студента: группы, оплаты, задолженность

### 📚 Группы (`/groups`)
- Управление группами (название, преподаватель, кабинет, стоимость)
- Расписание: чётные / нечётные дни
- Зачисление и отчисление студентов

### 🗓 Расписание (`/schedule`)
- Календарный вид расписания по месяцам
- Визуальное разделение чётных и нечётных дней
- Отображение групп, преподавателей, кабинетов

### 👨‍🏫 Преподаватели (`/teachers`)
- Список преподавателей
- Создание, редактирование, удаление

### 💼 Менеджеры (`/managers`)
- Список менеджеров
- Привязка студентов к менеджерам

### 💳 Оплаты (`/payments`)
- Регистрация оплат студентов
- Фильтрация по студенту
- Описание платежа

### 📋 Посещаемость (`/attendance`)
- Отметка посещаемости по группам
- **Месячный журнал** — таблица студенты × дни
- Разделение на чётные / нечётные дни (цветовая маркировка)
- **Комментарии** к каждой записи (причина отсутствия)
- Процент посещаемости по студенту за месяц

### 📊 KPI и Аналитика

#### KPI Менеджеров (`/kpi/managers`)
- Выручка, задолженность, конверсия
- Прогресс-бары

#### KPI Преподавателей (`/kpi/teachers`)
- Посещаемость, удержание студентов, задолженность
- Прогресс-бары

### 📈 Статистика (`/statistics`)
- **Посещаемость**: за неделю, месяц, квартал, год
- **Финансы**: общая выручка, средний чек, топ должников
- **Рейтинг менеджеров**: по конверсии и выручке
- **Рейтинг преподавателей**: по посещаемости и удержанию
- **Рейтинг студентов**: топ по оплатам и посещаемости
- Выбор года для исторических данных

## 🎨 Дизайн

- **Тёмная тема** (по умолчанию): тёмные карточки, фиолетовый акцент
- **Светлая тема**: светлый фон, те же акценты
- Переключатель в боковой панели (☀ / 🌙)
- Сохранение выбора в `localStorage`
- Адаптивный дизайн для мобильных устройств
